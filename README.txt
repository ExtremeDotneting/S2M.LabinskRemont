Инструкция по сборке веб проекта:
выполнить один раз{
	1. Для сборки вам сначала нужно установить Node.js вместе с npm. Также может потребоваться установка python 2.7.
	2. Запустите cmd.
	3. Перейдите в папку FakeCasino.Web при помощи команды pushd <путь к папке>
	4. Введите npm install для загрузки зависимостей. Если возникает ошибка с Visual C++ 2005 , то сначала введите npm install --global --production windows-build-tools
}выполнить один раз
	5. В файле FakeCasino.Web/static/global.conf.js вы можете настроить окружение (нативное или веб-сайт) и задать ссылки для редиректа.
	6. Введите в консоли npm run build. Результаты сборки ищите в папке FakeCasino.Web/dist.
	7. Для развертывания скопируйте все содержимое папки dist на ваш php сервер.

Инструкция по сборке андроид приложения на с#:
	1. Сначала собирите веб-проект.
	2. Скопируйте содержимое папки FakeCasino.Web/distв папку FakeCasino/Assets .
	3. Откройте проект в visual studio. Скачайте android sdk. Я использовал для android7.1
	4. Соберите проект в режимер релиза. 
	5. Заархивируйте проект (пкм по названию проекта и дальше по пунктам).

Инструкция по сборке андроид приложения на PhoneGap:
	1. Сначала собирите веб-проект.
	2. Скопируйте содержимое папки FakeCasino.Web/dist в папку FakeCasino.PhoneGap . Файл config.xml не удалять.
	3. Удалите папки static/php , static/img
	4. Создайте гит репозиторий, закомитте изменения. Проще всего сделать это используя Git Extensions.
	5. Создайте гитхаб репозиторий и залейте туда проект.
	6. Авторизируйтесь на https://build.phonegap.com/apps , укажите ссылку на гитхаб репозиторий и следуйте инструкции по сборке.
